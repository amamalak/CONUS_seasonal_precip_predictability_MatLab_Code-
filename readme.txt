The file "USWprecip_prediction_COBEv2.m" is the main code script for the study by Mamalakis et al. (2022).
This code is written in MatLab.

Citation:

Mamalakis, A., A. AghaKouchak, J.T. Randerson, E. Foufoula-Georgiou (2022) Hotspots of predictability: Identifying regions of high
precipitation predictability at seasonal timescales from limited time series observations, Water Resources Research.
