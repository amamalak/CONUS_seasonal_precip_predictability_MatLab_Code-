The file "USWprecip_prediction_COBEv2.m" is the main code script for the study by Mamalakis et al. (2022).
This code is written in MatLab.
The exact data sources used for the study are free online and urls are provided in the script.
However, if you want to use different data (or updated versions of it), you may need to modify certain parts of the script accordingly. 
Please make sure you cite this study if you use parts of this code or the methodology that is based on. 

Citation:

Mamalakis, A., A. AghaKouchak, J.T. Randerson, E. Foufoula-Georgiou (2022) Hotspots of predictability: Identifying regions of high precipitation predictability at seasonal timescales from limited time series observations, Water Resources Research.
